# Claude 工作規範

## 輸出規範

### 分支資訊
- 每次提交或推送後，**必須告知使用者當前分支名稱**
- 格式範例：
  ```
  **分支**: `claude/feature-name-xxxxx`
  **檔案位置**: `path/to/file.md`
  ```

### 文件變更
- 提交後需提供：
  - 分支名稱
  - 檔案路徑
  - 變更摘要

## 專案慣例

### PRD 文件
- 位置：`backend/docs/`
- 命名：`PRD_<功能名稱>.md`
- 版本記錄：每次更新需更新版本號和變更記錄

### Git 提交訊息
- 使用繁體中文
- 格式：`<type>: <描述>`
- 類型：`feat`, `fix`, `docs`, `chore`, `refactor`
